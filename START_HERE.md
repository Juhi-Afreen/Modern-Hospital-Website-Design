# Start Here: Hospitalia Website Project

Welcome to the **Hospitalia** project! This document acts as your entry point for understanding the codebase and getting started with development.

## 🎯 Project Overview

This is a modern, responsive healthcare website template built with:

- **HTML5**: Semantic and accessible structure.
- **Tailwind CSS**: Utility-first styling (configured via JS).
- **GSAP (GreenSock)**: High-performance animations for scroll effects and interactions.
- **Lucide Icons**: Clean, lightweight SVG icons.

## 📂 Key Directories

- **`index.html`**: The main landing page.
- **`css/`**: Contains custom overrides (`styles.css`).
- **`js/`**: Contains logic (`main.js`) and configuration (`config.js`).
- **`assets/`**: Images and static media.

## 🚀 First Steps

1. **Open `index.html`** in your browser to view the site.
2. Read **`QUICK_START.md`** for immediate setup instructions.
3. Review **`PROJECT_STRUCTURE.md`** to understand how files are organized.
