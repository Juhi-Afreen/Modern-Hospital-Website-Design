# Hospitalia - Modern Healthcare Website

A premium, responsive website template designed for hospitals, clinics, and healthcare providers. It features smooth scroll animations, a clean UI, and a fully mobile-responsive design.

## ✨ Features

- **Responsive Navigation**: Mobile-friendly menu with backdrop blur.
- **Hero Section**: Animated shapes and call-to-action buttons.
- **Service Stack**: Sticky stacking card animation for services.
- **Departments Grid**: Hover effects and grid layout for medical departments.
- **FAQ Accordion**: Smooth expand/collapse functionality.
- **Appointment Form**: Styled input fields for patient booking.

## 🛠 Tech Stack

- **HTML5**
- **Tailwind CSS** (via CDN script)
- **GSAP** (ScrollTrigger & Core)
- **Lucide Icons**
