# Quick Start Guide

Get up and running with the Hospitalia website in minutes.

## Prerequisites

- A modern web browser (Chrome, Firefox, Edge, Safari).
- A code editor (VS Code recommended).
- _Optional_: A local server (e.g., Live Server extension for VS Code) is recommended for best performance with animations.

## Running the Project

1. **Clone or Download** the repository.
2. **Open the folder** in VS Code.
3. **Right-click `index.html`** and select **"Open with Live Server"** (if installed).
   - _Alternatively_, double-click `index.html` to open it directly in your browser.

## Customization Basics

- **Colors**: Open `js/config.js` to modify the Tailwind color palette (e.g., `primary`, `teal`).
- **Content**: Edit `index.html` to change text, images, and links.
- **Styles**: Add custom CSS to `css/styles.css`.

## Common Issues

- **Icons not showing?** Ensure you have an internet connection to load scripts from CDNs.
- **Animations laggy?** Check `js/main.js` and ensure GSAP is loaded correctly.

You're ready to go! 🏥
